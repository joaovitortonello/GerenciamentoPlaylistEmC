# GerenciamentoPlaylistEmC
  Atividade desenvolvida em 2 semanas 02/06/2026 - 15/06/2026, Consiste na criação de um sistema de gerenciamento de músicas, aonde o usuário tem acesso a uma playlist pré-definido e consegue personalizar a sua playlist se baseando nela.
  Foi utilzado funções do básico (Scanf, Switch Case, If, Else...), Intermediário (While, Ponteiro Simples, Do While...) até considerado por mim como avançado como Manipulação de Lista e Manipulação de Arquivo. 
  O programa ficou considerávelmente completo acredito que se inserisse uma forma de salvamento da playlist ficava mais realista, tentei, porém sem sucesso tive de entregar o trabalho assim mesmo, futuramento gostaria de reformar esse projeto com mais tempo.
